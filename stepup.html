Redesigning StepUp app

Project Needs

Redesign StepUp app to increase customer retention, make it more visually appealing and introduce new features with minimum changes to the existing app.

UX Methodologies

  • Comparative/Competitive Analysis
  • Affinity diagrams
  • Persona creation
  • Task Analysis
  • Concept Map
  • Userflow
  • Wireframing
  • Prototype testing in Invision

Software Used

  • Sketch
  • Invision
  • Keynote Presentation

Research

The StepUp team had given me access to their Facebook Analytics page from which the primary research was done. For competitive analysis two Pedometer ++ and Fitbit were main considerations as per the request of the Stakeholder.
